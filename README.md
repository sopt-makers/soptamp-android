<h2> What is SOPTAMP? </h2>

SOPT는 IT와 벤처창업에 뜻이 있는 대학생들이 모인
[국내 최대 규모의 대학생 연합 IT 벤처창업 동아리](https://sopt.org/)입니다.

SOPT 공식 앱에서는 솝트인이라면 꼭! 즐겨야 하는 활동들을 미션 형태로 제시합니다.

미션을 달성하며 솝트에서의 추억을 보관하고
랭킹을 올리며 경쟁하는 재미를 느껴보세요.
